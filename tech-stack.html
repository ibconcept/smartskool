SmartSkool Tech Stack

Frontend

  • React.js: JavaScript library for building user interfaces
  • Next.js: Framework for server-side rendering and static site generation
  • Material-UI / Ant Design: Pre-styled component libraries for responsive design

Backend

  • Node.js with Express.js: Fast and flexible backend framework
  • GraphQL: Efficient data fetching for complex queries
  • Django / Flask: Alternative Python frameworks for specific features or microservices

Database

  • PostgreSQL: Powerful relational database system
  • MongoDB: NoSQL database for handling unstructured data

Authentication & Authorization

  • Auth0 / Firebase Authentication: Comprehensive authentication solutions
  • JWT: Secure token-based authentication

Real-time Features

  • Socket.io: Real-time communication

Hosting & Deployment

  • Vercel / Netlify: Deployment for frontend
  • Heroku / DigitalOcean: Backend deployment
  • AWS: Robust cloud services

CI/CD

  • GitHub Actions: Automating build, test, and deployment pipelines
  • Jenkins: Open-source automation server for CI/CD

Additional Tools

  • Docker: Containerizing the application
  • Kubernetes: Orchestrating Docker containers
  • ElasticSearch: Advanced search capabilities
